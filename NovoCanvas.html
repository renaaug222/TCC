TAREFA DE PREVISÃO

A tarefa de machine learning foi a classificação de severidade de acidentes entre 'Fatal e Não Fatal'

As variáveis preditoras iniciais foram Tempo, Pavimento, Sexo, Cinto Segurança, Embriaguez, Tipo Acidente, Número Bairro

A variável alvo utiliada foi Severidade;

Dos algoritmos testados, a Árvore

de Decisão, foi o que apresentou mais alta acurácia.

 


DECISÃO

O resultado do modelo é a predição se um determinado acidente foi fatal, subsidiando providências do Governo Federal para evitar novos incidentes.

PROPOSTA DE VALOR

Para implantação de políticas públicas em diversas áreas é essencial ter dados que sirvam de subsídio para tomadas de decisões mais assertivas. O objetivo deste trabalho é permitir avaliação dos impactos de diversos itens que compõem um acidente na severidade do resultado dele, auxiliando o Governo Federal em ações de prevenção de acidentes.

Este estudo visa exatamente buscar e comparar três dos mais conhecidos modelos de machine learning: Regressão Linear, Árvores de Decisão e Naive Bayes. Estabelecendo qual dentre eles obteve o melhor desempenho como ferramenta de classificação

FONTES DE DADOS

Para o presente estudo foram utilizadas três bases de dados da Prefeitura de Belo Horizonte que fazem parte do Portal Brasileiro de Dados Abertos.

IMPACT SIMULATION

Para simulação foi utilizada a validação cruzada. Para os três modelos implementados, foram avaliadas as seguintes métricas (através do node Scorer), que permitiram a escolha de um que obteve melhor desempenho

  1. Matriz de Confusão;
  2. Recall;
  3. Precisão;
  4. Sensibilidade;
  5. Especificidade;
  6. F-measure;
  7. Acurácia;
  8. Cohen's Kappa;

FAZENDO PREVISÕES

As predições são realizadas após o tratamento, enriquecimento e o processamento dos dados e de forma muito rápida.

Para novas predições é importante atenção para a quantidade de opções (domínios) para as variáveis preditoras evitando problemas na aplicação dos modelos. Além disso, é necessária avaliação e tratamento da base que, para este estudo, estava desbalanceada.

MODELOS DE CONSTRUÇÃO

Foram testados três algoritmos: Regressão Linear, Naive bayes e Árvore de Decisão

Os três modelos foram construídos na ferramenta Knime, porém o que apresentou o melhor desempenho na classificação foi a Árvore de Decisão.

FEATURES

As variáveis preditoras são:  

  • Tempo
  • Pavimento
  • Sexo
  • Cinto Segurança
  • Embriaguez
  • Tipo Acidente
  • Número Bairro

A variável alvo é Severidade;

MONITORING

O modelo apresentado neste estudo ainda não foi implantado em produção para que haja um monitoramento.
